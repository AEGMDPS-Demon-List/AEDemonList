# Pointercrate Clone

A starter React + Vite project scaffold for replicating a leaderboard-style site like `pointercrate.com`.

## Setup

1. Install dependencies:
   ```bash
   npm install
   ```
2. Start development server:
   ```bash
   npm run dev
   ```
3. Open the local URL shown in the terminal.

## Notes

- The app is intentionally lightweight and uses React state for navigation.
- Extend it with routing, backend APIs, authentication, and data persistence as needed.
